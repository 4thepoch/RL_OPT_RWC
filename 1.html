Module Temperature Prediction Model

  • 알고리즘 구축 시기는 계절상 겨울이기에 실제 현장 테스트가 불가한 상황에서 온라인 상 제어 시뮬레이션을 수행하기 위해, 가상의 우수제어 환경 모델을 구축이 필수적이라 판단
  • 기존에 축적된 데이터와 살수 데이터를 기반으로 환경 모델 구축
  • 우수제어로 인해 변형되는 데이터는 모듈온도와 순시전력이기 때문에 주변 데이터를 기반으로 모듈온도 및 순시전력을 예측할 수 있는 모델이 필요
  • 살수 시 / 방치 시 모듈온도를 실시간 예측하고 살수효과로 인해 모듈온도가 하강하였다가 다시 방치 시 모듈 온도와 동일해지는 시점을 살수효과 종료 시점으로 설정하는 프로세스가 학습 환경 모델에 적용되어져야 하며 이를 CEET Decision Model(Control Effect End Time Decision Model; 제어효과 종료 시점 결정 모델)로 정의했으며 상세 내용은 아래와 같음
